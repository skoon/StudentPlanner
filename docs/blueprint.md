# **App Name**: Student Planner Pro

## Core Features:

- Assignment Tracking: Allow students to input and track assignments with due dates and descriptions.
- Test and Project Scheduling: Enable students to schedule tests and projects, setting reminders for upcoming deadlines.
- Calendar View: Provide a calendar view that shows all assignments, tests, and projects in one place.
- Prioritization Assistance: Suggest a prioritized list of tasks based on urgency and importance using AI analysis of due dates and assignment types.
- Reminders: Send notifications for upcoming deadlines and schedule changes.
- Subject Categories: Allow students to organize their assignments by subject or course.
- Pomodoro Timer: A timer to facilitate studying using the Pomodoro technique.

## Style Guidelines:

- Primary color: A calming blue (#6699CC) to promote focus and concentration.
- Background color: Light gray (#E5E7E9) to ensure readability and reduce eye strain.
- Accent color: A vibrant yellow (#FFB347) for important deadlines and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif to make it easy to read and modern.
- Use clear, minimalist icons to represent different types of assignments and subjects.
- A clean, intuitive layout with a focus on easy navigation and information hierarchy.
- Subtle transitions and animations to provide feedback and guide the user through the app.
